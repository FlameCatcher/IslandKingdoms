# IslandKingdoms


I started to create this game because i wanted to create a 2D game in which you can create your own custom kingdom and battle with other custom AI kingdoms.

The ideea started after I played a game in which i was a commoner in a medieval land and i had to rise to the rank of a King and manage and expand my kingdom. Altho the game was fun, it felt a little lackluster because there was no way watsoever to customize the kingdom so it kinda felt rigid and predetermined.
